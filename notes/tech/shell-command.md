# Bash快捷方式

## 快捷键

- 光标移动到行首：`CTRL-A`
- 光标移动到行尾：`CTRL-E`
- 光标向前移动一个字符：`CTRL-B`
- 光标向后移动一个字符：`CTRL-F`
- 退格：`CTRL-H`
- 删除一个单词：`CTRL-W`
- 删除行：`CTRL-U`
- 撤回上面两个操作：`CTRL-Y`
- 文件结尾EOF：`CTRL-D`
- 清屏：`CTRL-L`
- 回车：`CTRL-M`，`CTRL-J`
- 回溯搜索历史：`CTRL-R`
- 撤销操作：`CTRL--`
- 在行首和光标两个位置间进行切换：`CTRL-XX`
- 上一条命令：`CTRL-P`

## 小技巧

- 开启`vi`模式：`set -o vi`
- 上一条命令：`!!`
- 查看本机IP：`curl ip.cn`
- 重置终端：`reset`
- 上一个目录：`cd -`
- 通过A跳板连接B：`ssh -t A ssh B`
