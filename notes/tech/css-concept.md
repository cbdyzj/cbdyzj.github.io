# CSS

### HTML

- 意义优先


- 滥用类（classitis），滥用div（divitus）

### 选择器

- 元素选择器，后代选择器

```css
span {}
h2 p {}
```

- ID，类，伪类

```css
#intro {}
.date {}
a:link {}
```

- 通用选择器（通配符）

```css
* {}
```

- 子选择器，兄弟选择器

```css
#nav>li {}
h2+p {}
```

- 属性选择器

```css
span[title] {}
a[rel="nofollow"] {}
```

### 层叠样式

选择器权重 = 1000 * a + 100 * b + 10 * c + d

1. a取决于是否是行内样式
2. b为ID选择器数量
3. c为类、伪类、属性选择器数量
4. d为元素择器、伪元素选择器数量
5. 样式会继承与覆盖
